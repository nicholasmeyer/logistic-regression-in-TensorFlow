# Logistic regression in TensorFlow

Implementation of logisitc regression in TensorFlow according to the project structure outlined by Danijar Hafner.

### Dependencies

```
TensorFlow
```

### Usage

To run the code 

```
python logit.py
```

Optional arguments 
```
-b (batch size)
-n (number of iterations)
-d (display frequency)
-l (learning rate)
```
